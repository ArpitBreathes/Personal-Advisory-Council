## üÜò Emergency Troubleshooting

### AI Not Responding

**Cause**: API key issue or rate limit

**Fix**:

1. Check console for errors
2. Refresh page
3. If persists, show pre-recorded video

### Internet Down

**Backup Plan**:

- Have 60-second video recorded
- Show screenshots instead
- Pivot to technical architecture explanation

### Demo Crashes

**Recovery**:

1. Say: "Let me show you how this works..." (stay calm)
2. Refresh browser
3. If won't load, go to backup video
